# 🚀 Руководство пользователя - Улучшенный загрузчик FAQ

## 📋 Краткое руководство

### Как запустить программу:

**🖱️ Самый простой способ:**
```
Дважды кликните на файл: Enhanced_FAQ_Loader.bat
```

**🐍 Через Python:**
```bash
python utils/enhanced_faq_gui.py
```

### Основные функции интерфейса:

## 1. 📁 Выбор и просмотр файлов

### Кнопки:
- **📄 Добавить файлы** - выбрать отдельные файлы для обработки
- **📁 Добавить папку** - выбрать папку (программа автоматически найдет подходящие файлы)
- **🧹 Очистить всё** - удалить все выбранные файлы из списка

### Таблица файлов показывает:
- **Имя** - название файла
- **Тип** - автоматически определенный тип файла
- **Размер** - размер в мегабайтах
- **Статус** - готов ли файл к обработке

## 2. ⚙️ Настройки обработки

### Опции:
- **🔄 Объединить с существующим FAQ** - добавить к имеющимся записям (рекомендуется)
- **💾 Создать резервную копию** - автоматическое резервирование (рекомендуется)
- **✅ Проверить содержимое** - проверка качества данных перед обработкой
- **📦 Группировать похожие записи** - умное объединение схожих записей
- **🧠 Умное слияние** - интеллектуальное разрешение конфликтов

## 3. 🚀 Основные действия

### Кнопки действий:
- **🚀 Обработать файлы** - запустить основную обработку выбранных файлов
- **👀 Просмотр FAQ** - показать текущее содержимое базы FAQ
- **✅ Проверить файлы** - предварительная проверка файлов без изменений
- **📋 Шаблоны** - создать файлы-шаблоны для упрощения подготовки данных
- **❌ Выход** - закрыть программу

## 4. 📊 Прогресс и логи

Нижняя панель показывает:
- **Прогресс-бар** - текущий прогресс выполнения
- **Лог операций** - подробный журнал всех действий с временными метками

## 💡 Советы по использованию

### ✅ Рекомендуемый рабочий процесс:

1. **Подготовка данных:**
   - Нажмите "📋 Шаблоны" для создания образцов
   - Подготовьте свои данные в CSV, Excel или JSON формате

2. **Добавление файлов:**
   - Используйте "📄 Добавить файлы" для отдельных файлов
   - Используйте "📁 Добавить папку" для массовой загрузки

3. **Предварительная проверка:**
   - Нажмите "✅ Проверить файлы" для проверки качества
   - Исправьте найденные ошибки при необходимости

4. **Обработка:**
   - Убедитесь что включены нужные настройки
   - Нажмите "🚀 Обработать файлы"
   - Дождитесь завершения (следите за прогресс-баром)

5. **Проверка результата:**
   - Используйте "👀 Просмотр FAQ" для проверки результата
   - Перезапустите бота для применения изменений

### 🎯 Поддерживаемые форматы файлов:

- **CSV файлы (.csv)** - таблицы с разделителями
- **Excel файлы (.xlsx, .xls)** - электронные таблицы
- **JSON файлы (.json)** - структурированные данные
- **Текстовые файлы (.txt)** - простые списки
- **Папки с файлами** - автоматическое создание FAQ из файловой структуры

### 🔍 Что программа делает автоматически:

- **Умное распознавание типов файлов** - анализирует не только расширение, но и содержимое
- **Поиск дубликатов** - находит и объединяет похожие записи
- **Проверка качества** - проверяет правильность структуры и содержимого
- **Резервное копирование** - создает копии перед изменениями
- **Детальное логирование** - записывает все операции для отслеживания

### ⚠️ Важные моменты:

1. **Резервные копии**: Всегда включайте "💾 Создать резервную копию"
2. **Проверка файлов**: Используйте "✅ Проверить файлы" перед основной обработкой
3. **Перезапуск бота**: После успешной обработки перезапустите бота командой или батч-файлом
4. **Формат данных**: Используйте шаблоны для правильного формата данных

### 🆘 Если что-то не работает:

1. **Проверьте лог** в нижней панели для деталей ошибки
2. **Проверьте формат файлов** используя функцию проверки
3. **Используйте шаблоны** для корректного формата данных
4. **Проверьте права доступа** к файлам и папкам

### 🎉 После успешной обработки:

Вы увидите сообщение об успехе с подробной статистикой:
- Количество добавленных записей
- Количество обновленных записей  
- Общее количество записей в FAQ
- Результаты проверки качества

**Не забудьте перезапустить бота для применения изменений!**
