# FAQ Bot - Quick Start Guide

## 🚀 Основные файлы запуска

Проект был упрощен и теперь использует всего 3 основных файла для управления:

### 1. `Clean.bat` - Очистка системы 🧹
**Назначение:** Полная очистка всех временных файлов, кэша и логов
**Когда использовать:**
- Перед первым запуском
- При проблемах с эмбеддингами
- После обновления FAQ
- При странном поведении бота

**Что очищается:**
- Кэш эмбеддингов (faq_embeddings.pkl, faq_index.faiss)
- Логи и временные файлы
- Python кэш (__pycache__)
- База данных аналитики
- Файлы разработки и тестирования

### 2. `Start.bat` - Запуск бота 🤖
**Назначение:** Основной файл для запуска FAQ бота
**Возможности:**
- Автоматическая активация виртуального окружения
- Проверка зависимостей
- Запуск бота с обработкой ошибок
- Опции перезапуска после остановки

**Перед запуском убедитесь:**
- Настроен файл `.env` с токеном бота
- Установлены все зависимости
- Виртуальное окружение создано

### 3. `FAQ_downloader.bat` - Управление FAQ 📚
**Назначение:** Интерфейс для загрузки и управления FAQ документами
**Возможности:**
- GUI интерфейс для простой загрузки
- Drag & Drop интерфейс
- Командная строка для продвинутых пользователей
- Создание шаблонов файлов
- Просмотр и статистика текущего FAQ
- Обновление эмбеддингов

## 📋 Порядок первого запуска

1. **Подготовка окружения:**
   ```batch
   setup.bat  # Если еще не запускали
   ```

2. **Настройка конфигурации:**
   - Скопируйте `.env.development` в `.env`
   - Добавьте ваш BOT_TOKEN в `.env`

3. **Загрузка FAQ (опционально):**
   ```batch
   FAQ_downloader.bat
   ```

4. **Очистка и запуск:**
   ```batch
   Clean.bat
   Start.bat
   ```

## 🛠️ Поддерживаемые форматы FAQ

### CSV файлы
```csv
query,answer,category
"Как работает бот?","Бот отвечает на вопросы из базы знаний","Общие вопросы"
```

### Excel файлы
- Первая строка - заголовки (query, answer, category)
- Поддержка .xlsx и .xls форматов

### Текстовые файлы
```
Вопрос 1^Ответ 1^Категория 1
Вопрос 2^Ответ 2^Категория 2
```

## 🔧 Устранение проблем

### Проблема: Бот не запускается
1. Запустите `Clean.bat`
2. Проверьте `.env` файл
3. Убедитесь что виртуальное окружение активировано
4. Запустите `Start.bat`

### Проблема: Не работает поиск FAQ
1. Запустите `FAQ_downloader.bat`
2. Выберите "Обновить эмбеддинги"
3. Перезапустите бота

### Проблема: Ошибки импорта
1. Активируйте виртуальное окружение
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите `Start.bat`

```
├── data/                  # Данные FAQ

```

## 💡 Советы

- Всегда запускайте `Clean.bat` перед важными обновлениями
- Используйте GUI интерфейс в `FAQ_downloader.bat` для простоты
- Регулярно создавайте резервные копии `data/faq.json`
- Обновляйте эмбеддинги после значительных изменений FAQ
